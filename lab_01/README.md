## Jak uruchomić program
W celu uruchomienia program należy otworzyc w środowisku programistycznym z zainstalowanym kompilerem Pythona i skompilować, ja zrobiłem to za pośrednictwem środowiska PyCharm. 
Moje środowisko automatycznie uruchamia program po kompilacji. W VS Code procedura powinna być taka sama.
